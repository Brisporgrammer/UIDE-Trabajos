## Juego Snake en Python
### Descripción:
Este proyecto consiste en el desarrollo del clásico juego Snake utilizando el lenguaje de programación Python y las librerías estándar turtle (para la interfaz gráfica) y random (para la generación aleatoria de la comida).
El programa permite al usuario controlar una serpiente que debe desplazarse por la pantalla para consumir alimento, aumentar su tamaño y acumular puntaje, evitando colisionar con las paredes o con su propio cuerpo.
### Objetivo del programa
Desarrollar un videojuego a elección, en este caso snake
•	Uso de estructuras de datos (listas)
•	Control de flujo
•	Funciones
•	Detección de colisiones
### Principales funcionalidades:
Movimiento de la serpiente.
Generación aleatoria de comida.
Sistema de puntaje.
Detección de colisiones.
